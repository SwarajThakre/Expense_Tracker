# 💰 Expense Tracker

A simple and interactive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript** that helps users track their income and expenses.

Users can add transactions, view their balance, and monitor spending in a clean and responsive interface. The application demonstrates core JavaScript concepts such as **DOM manipulation, event handling, and local storage**.

---

## 🚀 Features

* ➕ Add income and expense transactions
* 📊 View total balance
* 💸 Track total income and expenses
* 🗑️ Delete transactions
* 💾 Data saved using **Local Storage**
* 📱 Responsive user interface

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Application logic and interactivity
